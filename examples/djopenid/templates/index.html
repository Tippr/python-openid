This is a Django package which implements an OpenID server and consumer, using the assumptions and protocol extensions leveraged by Tippr for partner login. It is based on the djopenid codebase from Janrain.

To begin, click one of these links:

  • Tippr (Relying Party)
  • Partner (Identity Provider)

Note: If you want to test the example consumer using the example server, you must start a separate server process for each application.
